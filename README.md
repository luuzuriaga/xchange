# X-Change | Conversor de Monedas Moderno

![X-Change Logo](img/favicon.svg)

X-Change es una aplicación web premium, ligera y moderna para la conversión de divisas en tiempo real. Diseñada con una estética fintech, ofrece una experiencia fluida tanto en escritorio como en dispositivos móviles.

## 🚀 Características Principales

-   **Conversión en Tiempo Real**: Los resultados se actualizan instantáneamente mientras escribes.
-   **Detección Automática de Contexto**: Identifica automáticamente tu moneda local (especialmente optimizado para **Soles Peruanos - PEN**) y establece el Dólar (USD) como destino predeterminado.
-   **Gráfica de Tendencia (30 días)**: Visualiza el comportamiento del mercado del último mes con una gráfica SVG dinámica y minimalista.
-   **Historial de Conversiones**: Guarda tus últimas 10 operaciones localmente para una consulta rápida.
-   **Diseño Premium**: Interfaz moderna con efectos de glassmorphism, modo oscuro integrado y animaciones suaves.
-   **Resiliencia**: Sistema de fallback local que permite que la aplicación funcione incluso si la API externa falla.

## 🛠️ Tecnologías Utilizadas

-   **HTML5 & CSS3**: Estructura semántica y diseño responsivo sin frameworks externos.
-   **Vanilla JavaScript**: Lógica pura para máxima velocidad y ligereza.
-   **APIs Externas**:
    -   [MoneyConvert API](https://moneyconvert.net/) para tasas en tiempo real.
    -   [Frankfurter API](https://www.frankfurter.app/) para datos históricos.
-   **SVG**: Gráficas generadas dinámicamente.
